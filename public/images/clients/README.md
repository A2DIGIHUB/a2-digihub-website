# Client Logos Directory

Place client/partner organization logos here.

## Guidelines:
- **Format**: PNG or SVG preferred
- **Size**: Recommended width: 120-150px
- **Background**: Transparent preferred
- **Naming**: Use lowercase with hyphens (e.g., `company-name.png`)

## Example:
```
clients/
  ├── organization-1.png
  ├── organization-2.svg
  ├── partner-company.png
  └── README.md (this file)
```

The logos will be displayed in the "Trusted by organizations across Africa" section on the homepage.
